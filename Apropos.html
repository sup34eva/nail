Le but de ce logiciel nail (editeur de monde) est de pouvoir crée un monde sans aucune restriction.
Pour vous deplacer dans l'environnement avec les flèches Haut, Bas, Gauche, Droite.
Vous pouvez ajouter plusieurs objet predefini en cliquant sur le menu objet qui ce trouve sur la droite de la fenetre.
Pour chaque objet crée vous pouvez modifier : le nom , la taille et la couleur de chaque objet.
Vous pouvez aussi modifier sa taille( largeur, longeur) tout ceci a l'aide du l'infobox qui ce situe en bas de votre écran.
Vous avez le choix de regrouper plusieurs objet dans une ou plusieurs groupe pour pouvoir les modifiers ou les deplacer en même temps.
Pour ceci il vous suffit de cliquer sur les vos noms d'objet et ensuite de cliquer sur grouper.
Dans le menu objet nous avons 3 bouton qui vous permettra d'ajouter un objet de votre choix en le selectionnant dans le menu deroulant.
de le supprimer en cliquant sur le bouton supprimer.

Vous disposez de 4 objet 2D :

  • Rectangle
  • Ligne
  • Cercle
  • Trapeze

Vous disposez de 4 objet 3D :

  • Cylindre
  • Cube
  • Pyramide
  • Sphere

Vous pouvez mettre le logiciel en plein ecran a l'aide du bouton Fullscreen.
Et le bouton ResetCam permet de centrer la camera a son point de depart.
vous pouvez aussi editer vos objet a l'aide de l'onglet edition dans le menu qui vous permettra de copier coller couper supprimer vos objets.
Vous pouvez aussi enregistrer votre monde, afin de pouvoir re-ouvrir est continuer plus tard grace a l'onglet fichier.
